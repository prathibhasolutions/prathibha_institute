Python MCQs and Programming Questions

Multiple Choice Questions (MCQs)

Programming Questions with Example Outputs

  1. Write a program to check if a number is even or odd.
    Example:
    Input: 5
    Output: Odd
  2. Write a program to print the factorial of a number using recursion.
    Example:
    Input: 4
    Output: 24
  3. Write a program to reverse a string entered by the user.
    Example:
    Input: hello
    Output: olleh
  4. Write a program to check if a string is palindrome.
    Example:
    Input: madam
    Output: Palindrome
  5. Write a program to find the sum of elements in a list.
    Example:
    Input: [1, 2, 3, 4]
    Output: 10
  6. Write a program to print Fibonacci series using a loop.
    Example:
    Input: 5
    Output: 0 1 1 2 3
  7. Write a program to count vowels in a given string.
    Example:
    Input: education
    Output: 5
  8. Write a program to print a pyramid pattern using nested loops.
    Example:
    Input: 3
    Output:
       *
      ***
     *****
  9. Write a function that returns the maximum of three numbers.
    Example:
    Input: 12, 45, 23
    Output: 45
  10. Write a program to remove duplicates from a list.
    Example:
    Input: [1, 2, 2, 3, 1]
    Output: [1, 2, 3]
