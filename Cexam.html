C Language MCQs and Programming


Multiple Choice Questions (MCQs)

Programming Questions with Example Outputs (Moderate Level)


  1. Write a program to check whether a given year is a leap year or not using if-else.
    Example:
    Input: 2024
    Output: Leap Year
  2. Write a program to print the factorial of a number using recursion.
    Example:
    Input: 5
    Output: 120
  3. Write a program to create a simple calculator using switch-case.
    Example:
    Input: 4 + 3
    Output: 7
  4. Write a program to print a pyramid pattern using nested loops.
    Example:
    Input: 3
    Output:
       *
      ***
     *****
  5. Write a program to count the frequency of each digit in a number.
    Example:
    Input: 122334
    Output: 1->1, 2->2, 3->2, 4->1
  6. Write a program to sort an array using the Bubble Sort algorithm.
    Example:
    Input: [4, 2, 1, 5]
    Output: [1, 2, 4, 5]
  7. Write a program to reverse a string without using strrev().
    Example:
    Input: hello
    Output: olleh
  8. Write a function to check whether a number is a prime number or not.
    Example:
    Input: 13
    Output: Prime
  9. Write a program to find the second largest number in an array.
    Example:
    Input: [10, 20, 15, 25]
    Output: 20
  10. Write a program to copy the contents of one file to another.
    Example:
    Input File: data.txt
    Output File: copied.txt
  11. Write a program to count vowels, consonants, digits, and special characters in a string.
    Example:
    Input: Hello@123
    Output: Vowels=2, Consonants=3, Digits=3, Special=1
  12. Write a program to merge two arrays into a third array.
    Example:
    Input: [1,2] and [3,4]
    Output: [1,2,3,4]
  13. Write a recursive function to find the nth Fibonacci number.
    Example:
    Input: 6
    Output: 8
  14. Write a program to read a file and count number of lines and words.
    Example:
    Input: file.txt
    Output: Lines: 5, Words: 27
  15. Write a program to check if a string is a palindrome using a function.
    Example:
    Input: radar
    Output: Palindrome
