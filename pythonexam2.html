Python MCQs and Programming Questions (Set 2)


Multiple Choice Questions (MCQs)

Programming Questions with Example Outputs


  1. Write a program to check whether a number is a multiple of 7.
    Example:
    Input: 21
    Output: Yes, multiple of 7
  2. Write a program to calculate Celsius to Fahrenheit using the formula: F = (C × 9/5) + 32.
    Example:
    Input: 25
    Output: 77.0
  3. Write a program to find the largest digit in a number.
    Example:
    Input: 2945
    Output: 9
  4. Write a program to check if the input string is lowercase.
    Example:
    Input: hello
    Output: True
  5. Write a program to find the sum of squares of first n natural numbers.
    Example:
    Input: 4
    Steps: 1² + 2² + 3² + 4² = 1 + 4 + 9 + 16 = 30
    Output: 30
  6. Write a program to print the multiplication table of a number.
    Example:
    Input: 3
    Output: 3 6 9 12 15 18 21 24 27 30
  7. Write a program to count the number of words in a sentence.
    Example:
    Input: I love Python
    Output: 3
  8. Write a program to print a diamond pattern using loops.
    Example:
    Input: 3
    Output:
      *
     ***
    *****
     ***
      *
  9. Write a function to count the number of prime numbers between 1 to n.
    Example:
    Input: 10
    Output: 4 (2, 3, 5, 7)
  10. Write a program to find the second largest number in a list.
    Example:
    Input: [4, 1, 9, 7]
    Output: 7
